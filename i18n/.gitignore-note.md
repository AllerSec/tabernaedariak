# Carpeta `i18n/` — fuente del generador, no se publica

Esta carpeta contiene la **fuente** del sitio multiidioma:

- `build.mjs` — generador estático (Node, sin dependencias).
- `templates/` — plantillas HTML con marcadores `{{clave}}`.
- `strings/` — diccionarios de traducción por idioma (`eu/es/en/fr.json`).
- `data/` — datos no traducibles (`site.json`) y catálogo (`catalogo.json`).

## Cómo regenerar el sitio

```bash
node i18n/build.mjs
```

Genera los árboles `/eu`, `/es`, `/en`, `/fr` en la raíz del proyecto, más `sitemap.xml`.

Opciones útiles durante el desarrollo:

```bash
node i18n/build.mjs --only=es --out=.build-check   # solo un idioma, a un dir temporal
```

> Editar SIEMPRE aquí, nunca el HTML generado. Un cambio de texto se hace una vez
> en `strings/*.json` y se propaga a las 26 páginas de ese idioma al regenerar.
